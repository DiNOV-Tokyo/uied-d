# uied物体検出プロジェクト
https://github.com/MulongXie/UIED  
[こちら](https://github.com/MulongXie/UIED)を参考に開発

## インストール
$ pip install -r requirements.txt

## Usage

### 1. 物体検出
```python
$ python run_single.py **
```
** は読み込むファイル名（拡張子を除く）

### 2. JSON->HEML/CSS変換

```python
$ python compile_merge_img.py **
```
** は読み込むファイル名（拡張子を除く）
